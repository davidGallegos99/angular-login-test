# Prueba Técnica – Programador Frontend Angular (Middle)

---

Código: FND–FRONT–MID
Duración sugerida: 2 horas
Objetivo: Evaluar conocimientos en Angular, modularidad, routing, guards, formularios reactivos y buenas prácticas.

---

# Descripción

- Se solicita desarrollar una pequeña aplicación Angular que permita:

- Login de usuario con validación básica de correo y contraseña.

- Protección de rutas mediante guards.

- Implementar almenos 1 interceptor.

- Redirección a un dashboard solo si el usuario está autenticado.

- Uso de formularios reactivos y validaciones dinámicas.

- Separación de responsabilidades usando servicios para autenticación y lógica de negocio.

- El proyecto debe seguir buenas prácticas de Angular, incluyendo modularidad, componentes reutilizables y manejo de rutas.

---
# Entrega

- Repositorio Git con todo el proyecto Angular.

- Debe poder ejecutarse con npm install y ng serve (o equivalente si usan standalone + Vite).
